Visual Studio

Select custom if you want to specify the path to Visual Studio manually, or if you only have an Express version installed. The Express versions do not set the environment variables Strider uses to detect the install location.

This must be the path to the root folder that Visual Studio is installed to.

Project or Solution

If no project or solution is specified, strider will attempt to find a solution in the project's default directory. If there is more than one solution file, an error will be thrown.
